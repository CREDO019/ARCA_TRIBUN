# Çorum FK Oyuncu Görselleri

Bu klasör yalnızca kullanım izni doğrulanmış local oyuncu fotoğrafları içindir.
Uygulama çalışma anında dış siteden oyuncu görseli çekmez.

Dosya adı için küçük harfli ve alt çizgili biçim kullanın:

```text
serdar_gurler.jpg
```

Yetkili görsel eklendikten sonra ilgili `squad.image_url` değerini local asset
yolu olarak tanımlayın:

```text
assets/images/players/corum_fk/serdar_gurler.jpg
```

Görsel eklenmeyen oyuncularda arma, forma numarası ve initials içeren
placeholder gösterilir.
